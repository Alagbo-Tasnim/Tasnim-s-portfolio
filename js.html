Loops Demo: += and \n

This demo shows two things:
Number += step -> increase a number
String += "text\\n" -> adds text to string and starts a new line 



    
    

        
Click to count (uses += on a number , and \n for lines )

        
        
        
        
Each click does count += step. We also do output += "Count is: " + count +
                "\\n" so each result prints on a new line

    


    
    

        
For Loop Runner

        
        
        
        
        

            Runs:
            for (let i = start; 1 < =end; i += step)

            and logs each i on its own line using \n.
        

    


    
    

        
While loop (Countdown)

        
        
        

            Runs:
            while (n > 0) { log(n); n -= 1; }
        

    


    
    

        
Bonus) Press Enter to +1

        
Press the Enter key anywhere on the the page to do count += 1 and log a new line
